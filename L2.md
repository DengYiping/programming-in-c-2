# Review: Pointers, Arrays, Values
Pointer is a memory address.
## Pointer Arithematics
```
arr[i] = *(arr + i) = i[arr]
```
## Multidimension array
C treats multidimension array as a single dimension array.
```
int arr[3][4];
arr = *arr = &arr[0][0] = arr[0]
```
